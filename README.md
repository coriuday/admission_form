## ✨ Features

- 📋 **Multi-field form** — Name, Email, Phone, Course, Country, State, Previous Education Details
- ✅ **Smart validation** — Powered by Zod + React Hook Form with real-time error feedback
- 🌍 **Conditional logic** — State field shown only for domestic applicants
- 🎨 **Animated UI** — Floating background icons and smooth transitions via Framer Motion
- 📊 **Google Sheets integration** — Submissions saved directly to a Google Sheet via Google Apps Script (Web App)
- ⚡ **Graceful fallback** — Works perfectly without Google integration configured (safe for demos)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher (for the underlying environment)
- [Bun](https://bun.sh/) installed locally

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/coriuday/admission_form.git

# 2. Install dependencies
bun install

# 3. Configure environment variables
cp .env.example .env.local
# Edit .env.local and fill in your values

# 4. Start the development server
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🔧 Environment Variables

Create a `.env.local` file in the root directory. See [`.env.example`](.env.example) for reference.

| Variable | Description |
|----------|-------------|
| `GOOGLE_APPS_SCRIPT_URL` | The Web App URL generated by Google Apps Script |

> **Note:** The Google Sheets integration is optional. Without it, the form will still work and submissions are logged server-side.

## 📊 Google Sheets Integration

To save form submissions directly to a Google Sheet, we use a Google Apps Script Web App.

1. **Create a new Google Sheet** and setup the following headers in Row 1:
   Timestamp | Full Name | Email Address | Phone Number | Selected Course | Country | State | Previous Education Details
2. Click **Extensions > Apps Script**.
3. Replace the code with a `doPost(e)` function that appends `e.parameter` properties to `sheet.appendRow()`.
4. Click **Deploy > New deployment**.
5. Select type: **Web app**.
6. Set "Execute as" to **Me** and "Who has access" to **Anyone**.
7. Copy the generated Web App URL and add it to `.env.local` as `GOOGLE_APPS_SCRIPT_URL`.
